# Quickstart: Native Ubuntu

## Instalar

```bash
sudo ./deploy/native/install-orthanc-native.sh
```

## Validar

```bash
sudo ./scripts/orthanc/orthanc-healthcheck.sh
```

## Reiniciar

```bash
sudo ./scripts/orthanc/orthanc-restart.sh
```

## Remover

```bash
sudo ./deploy/native/purge-orthanc-native.sh
```
